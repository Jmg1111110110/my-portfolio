# 전민규 포트폴리오

이 저장소는 자기소개, 학습 내용, 자율 컨텐츠, 프로젝트 경험, 향후 계획 등을 정리한 포트폴리오입니다.

## 포트폴리오의 목적

이 포트폴리오는 학습 과정과 성장 기록을 정리하고, 관심 분야와 기술 스택, 프로젝트 경험을 체계적으로 보여주기 위해 제작되었습니다.

완성된 결과물뿐 아니라 학습 중인 내용과 앞으로의 계획도 함께 담아 성장 가능성을 나타내고자 합니다.

## 저장소 구성

- `intro.md` : 자기소개 파일
- `contents/content1.md` : 자율 컨텐츠 1 - 참여 팀프로젝트 소개 (BOSS)
- `contents/content2.md` : 자율 컨텐츠 2 - Tagnation 프로젝트 기획
- `index.html` : 메인 포트폴리오 페이지
- `style.css` : 포트폴리오 디자인
- `script.js` : 카드 클릭 기능 구현

## 주요 내용

- 자기소개
- 관심 분야
- 학습한 기술
- 프로젝트 경험
- 학습 기록
- 향후 계획

## 자율 컨텐츠 소개

### 자율 컨텐츠 1
BOSS(Basic Open Source Software) 프로젝트에 대한 소개와,  
프로젝트에서 제가 맡았던 백엔드 개발 역할 및 구현 내용을 정리하였습니다.

### 자율 컨텐츠 2
Tagnation이라는 NFC 기반 기부 굿즈 플랫폼 아이디어를 기획하고,  
서비스 목적, 주요 기능, 기술 스택, 기대 효과를 정리하였습니다.

## 목표

이 포트폴리오를 통해 지금까지의 학습 경험을 정리하고,  
앞으로 더 다양한 프로젝트를 통해 개발 역량과 협업 능력을 함께 성장시키고자 합니다.
