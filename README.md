# 전민규 포트폴리오

안녕하세요. 동국대학교 wise캠퍼스 컴퓨터공학과 2학년에 재학 중인 전민규의 GitHub 포트폴리오입니다.

이 저장소는 자기소개, 학습 내용, 자율 컨텐츠, 프로젝트 경험, 향후 계획 등을 정리한 포트폴리오입니다.

## 저장소 구성

- 자기소개 파일: `intro.md`
- 자율 컨텐츠 1: `contents/content1.md`
- 자율 컨텐츠 2: `contents/content2.md`
- 메인 포트폴리오 페이지: `index.html`

## 포트폴리오의 목적

이 포트폴리오는 저의 성장 과정, 학습 기록, 관심 분야, 기술 스택, 프로젝트 경험을 정리하여 보여주기 위한 공간입니다.  
완성된 결과물뿐 아니라 학습 중인 내용과 앞으로의 계획도 함께 담아 현재의 역량과 발전 가능성을 표현하고자 합니다.
